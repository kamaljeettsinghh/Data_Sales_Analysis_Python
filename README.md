# Data_Sales_Analysis_Python 
Project_Structure

## Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand
customer purchasing behavior and extract business insights.

## Dataset
- File: Diwali Sales Data.csv
- Format: Excel
- Contains customer demographic and purchase information

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Analysis Performed
- Data cleaning and preprocessing
- Gender-wise sales analysis
- Age group & occupation trends
- State-wise revenue distribution
- Product category insights
